Load data from csv

The source data is organized such that each State is a row, and then "sub-tables" are actually organized in the same Sheet for some reason. In the first grouped "sub-table," each Column is the public school expenditure (thousands of dollars) for a given year, starting from 1993 and running through 2016. The second "sub-table" contains student enrollment. The third "sub-table" contains per-pupil expenditures (dollars) for the same date range.

First, specify import options

opts = spreadsheetImportOptions;
v = strsplit(sprintf('Y%d-',1993:2016),'-'); % Variable names must start with a letter
opts.VariableNames = v(1:24);
opts.VariableTypes = repmat("double",1,24);
opts.RowNamesRange = "A5:A56";

Create the first table: Public school expenditures (thousands of dollars)

opts.DataRange = "B5:Y56";
G = readtable("public-school-per-pupil-expenditures.xlsx",opts);
G.Properties.UserData = struct('type','Gross','t',(1993:2016)'); % Add metadata to make easier to work with
G.Properties.Description = "Public school expenditures (thousands of dollars) by Year and State";

Create the second table: Enrollment (total count)

opts.DataRange = "AA5:AX56";
E = readtable("public-school-per-pupil-expenditures.xlsx",opts);
E.Properties.UserData = struct('type','Enrollment','t',(1993:2016)'); % Add metadata to make easier to work with
E.Properties.Description = "Public school enrollment by Year and State";

Create the third table: Per-pupil expenditures (dollars)

opts.DataRange = "AZ5:BW56";
T = readtable("public-school-per-pupil-expenditures.xlsx",opts);
T.Properties.UserData = struct('type','Per-Pupil','t',(1993:2016)'); % Add metadata to make easier to work with
T.Properties.Description = "Public school expenditures per-pupil (dollars) by Year and State";

We want to see trends in expenditures by Year

We'll create a graphic that has Years on the x-axis, total enrollment (counts) on the left y-axis, and expenditures per-pupil (dollars) on the right y-axis. The graphic will show +/- 1 standard deviation on the per-year trends so we can get an idea of how variable the data is across states, through time. This helps us understand the fundamental question: are we seeing a reduction in total cost per-student due to a stagnation of expenditure from a given administration, or are we seeing an increase in enrollment without an appropriate corresponding increase in
[~,ax] = plotTrendsByYear(E,T); % Do it as a separate function because *.mlx and git don't always play nicely
